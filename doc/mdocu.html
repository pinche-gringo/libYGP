NAME

mdocu - Extracts the function comments out of a (C++) source

SYNOPSIS

  mdocu [OPTIONS] [sourcefile [...]]

OPTIONS

  --verbose, -v .... Be verbose
  --help, -?, -h ... Show this help and exit

ARGUMENTS

  sourcefile ... Sources to inspect; if missing read from stdin

DESCRIPTION

Script to extract the function description out of a C(++) source. Those descriptions are identified by a C++ comment starting directly with Purpose: (no whitespaces in between the comment characters and the keyword).

Then the next lines starting with the C++ comment characters are checked for the keywords Parameters, Returns, Requires and/or Throws.

Their respective contents is printed out in a HTML format.
