Class XDialog

Derived from Gtk::Dialog
Derived by: BrowserDlg XAbout XDate XPrintDialog

#include <XDialog.h>

Description

Baseclass for dialogs; Installs OK and/or Cancel buttons and registers a command handler for them. Pressing OK is reported with the okEvent- method; cancel with the cancelEvent-method. The baseclase closes the dialog in both cases.

Pressing ESC is equal to selecting the CANCEL button.
