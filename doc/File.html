bool isExecuteable () const

Returns true, if the file is executeable. Under UNIX it checks the file-permissions (any execute-right is sufficient), under DOS/WINDOZE the extension of the file (EXE/COM/BAT).

Returns
bool: True, if the file is executeable
Remarks
- VBS, WSH, and whatever Windows executes directly are not considered executable
- The call is only valid after a successfull find.
bool isUserExec () const

Returns true, if the file is executeable for the actual user. Under UNIX it checks the rights for the effictive user under DOS/WINDOZE it the extension of the file (EXE/COM/BAT).

Returns
bool: True, if the file is executeable by the current user
Remarks
- VBS, WSH, and whatever Windows executes directly are not considered executable
- The call is only valid after a successfull find.
bool isHidden () const

Returns true, if the file is hidden. Under UNIX it checks if the file starts with a dot (.), under DOS/WINDOZE it inspects the attributes.

Returns
bool: True, if the file is hidden
Remarks
The call is only valid after a successfull find.
