Class ANumeric

Derived from: AttributValue

Header-file: ATNumeric.h

Description

Class for numeric values. As every AttributValue is supports undefined values.

Depending on the system and the available libraries the numeric value are stored as longs or with the GNU Multi Precission library (where numbers are of infinite length).

In- & output, conversion

virtual void readFromStream (istream& in)
Reads a formatted numeric value from a stream. If the input is not valid, the object is set to undefined.

Leading whitespaces in the stream are skipped as are thousand-seperaters in the number.

Parsing is stopped at EOF or at any non-digit (except thousand-seperators inside a number).
