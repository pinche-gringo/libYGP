Class ANumeric

Derived from: AttributValue
Derived by: none

#include <ANumeric.h>

Description

Class for numeric values. As every AttributValue is supports undefined values.

Depending on the system and the available libraries the numeric value are stored as longs or with the GNU Multi Precission library (where numbers are of infinite length).

virtual void readFromStream (istream& in) throw (invalid_argument)

Reads an unformatted numeric value from a stream. If the input is not valid, an excpetion is thrown.

Leading whitespaces in the stream are skipped.

Parsing is stopped at EOF or at any non-digit.
