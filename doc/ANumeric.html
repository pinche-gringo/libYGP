Class ANumeric

Derived from: AttributValue

Header-file: ATNumeric.h

Description

Class for numeric values. As every AttributValue is supports undefined values.

Depending on the system and the available libraries the numeric value are stored as longs or with the GNU Multi Precission library (where numbers are ov infinite length).
