Class extStream<T>

Derived from: private extStream, public T

Header-file: extStream.h

Description

Extended stream, designed to parse text.

It overcomes two drawbacks of the original stream:

  • Column- and line-information (which - to be fair - wouldn't make very much sense in the common stream anyway).
  • A variable-sized putback-buffer. The common stream only stores one block (and a few bytes from the last), so a pushback beyond this border is not possible.

Note: The extensions of the stream must be initialized with the init-method!

For easier usage there are definitions of the (propably) most common used extended Streams:

  • typedef extStream<istream> Xistream
  • typedef extStream<istream> Xifstream

void init ()
Initializes the extended stream. This call must be performed before the first data is read (or else the additional column/line does not match the reality).

Accessing the object

unsigned int getLine () const
Returns the actual line of the stream. This value is 0 before any data is read and starts with 1 for the first line (after the first read-operation). A new line starts after a character 10 (0x0a, or the C-charachter '\n'), which is UNIX-style but works also for the DOS/Windoze and the like.
unsigned int getColumn () const
Returns the actual position in the line of the stream. This value is 0 before any data (of the line) is read and starts with 1 after the first read-operation in the line (A new line starts after a character 10 (0x0a, or the C-charachter '\n'), which is UNIX-style but works also for the DOS/Windoze and the like.
