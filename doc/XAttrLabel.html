Class XAttributeLabel

Derived from Gtk::Label
Derived by: none

#include <XAttrLabel.h>

Description

Class to display an attribute in a formated style while preservering its other abilities.

It exists in two variants: XAttributeLabel stores a reference to the attribute to display, XAttributeLabel2 stores the attribute itself.

This class is designed to be used by AttributeValues (and derived types), but there exists specializations for:

  • short
  • unsigned short
  • int
  • unsigned int
  • long
  • unsigned long
  • double
