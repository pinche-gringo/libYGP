Class XFileEntry

Derived from Gtk::Entry
Derived by: none

#include <XFileEntry.h>

Description

Enhances the ordinary entry field with completing the input to the first file matching it (if the entered part is a valid file); analogue to modern web browsers.

During construction the type of files to complete can be specified. See the ocumentation of IDirectorySearch for details.
