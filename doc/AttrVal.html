Class AttributValue

Derived from: None

Header-file: AttrVal.h

Description

Base-class of all attributes. Attributes can have a value (according to their type) or be undefined (having no value at all).

An attribute is considered defined, if a value is explicitely set or if the defined ()-member is called (where the attribute get a default-value; depending on its type) -> every derived object must call AttributValue::define () when setting a value!
