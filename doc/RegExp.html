Class IRegularExpression

Derived from none
Derived by: FileRegularExpr RegularExpression

#include <RegExp.h>

Description

Base-class for any regular expression. Provides interface for other regular-expression-like classes.

Note: The pExpression-parameter is stored as is (and not copied); so take care it is valid during the life-time of the object.
