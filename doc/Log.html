Class Syslog

Derived from none
Derived by: none

#include <Log.h>

Description

Class to perform some logging into either a log-file (if available) or to console.

Every entry into the log-file can be classified (according to its serverity).

The following classifications are available:

EMERGENCY
The message says the system is unusable.
ALERT
Action on the message must be taken immediately.
CRITICAL
The message states a critical condition.
ERROR
The message describes an error.
WARNING
The message is a warning
NOTICE
The message describes a normal but important event.
INFO
The message is purely informational.
DEBUGGING
The message is only for debugging purposes.

The following macros (all of them taking just a character-pointer as parameter) exists to make the use of the class more easy:

LOGEMERGENCY
Logs an emergency-message./dd>
LOGALERT
Logs an alert-message.
LOGCRITICAL
Logs a critical message.
LOGERROR
Logs an error-message.
LOGWARNING
Logs a waring-message.
LOGINFO
Logs an informal message.
LOGDEBUGGING
Logs a debugging-message
