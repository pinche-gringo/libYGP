Member descriptions

Constructors & assignment

ATimestamp () - Defaultconstructor; the timestamp is not defined

ATimestamp (const ATimestamp& other) - Copyconstructor

ATimestamp (bool now) - Constructor; depending on the parameter the timestamp is either set to the 1st of January, 1900 0:00:00 (now = false), or to the current time (now = true).

ATimestamp (char Day, char Month, int Year, char Hour, char minute, char second) - Constructor; sets the passed time. The object is undefined, if the passed parameters represent no valid date/time-combination (e.g. Hour > 23, minute > 59, second > 61 or day > 31).

ATimestamp (const char* pTime) - Constructor; sets the passed time, which must be represented as Day.month.year hour:minute:second. If the buffer does not represent a valid time, the object is set to undefined. A NULL-pointer as parameter is not permitted!

virtual ATimestamp (std::string& time) - Constructor; sets the passed time, which must be represented as Day.month.year hour:minute:second. If the string does not represent a valid timestamp, the object is set to undefined.

ATimestamp (const struct tm& tm) - Constructor; assigns the values from the passed struct tm to the object.

ATimestamp (const time_t stamp) - Constructor; sets date & time to the value represented by the parameter.

virtual ~ATimestamp () - Destructor

ATimestamp& operator= (const ATimestamp& other) - Assignment-operator

ATimestamp& operator= (const char* pStamp) - Assignment-operator from an const char-pointer. The time must be passed as hour:minute:second. If the buffer does not represent a valid time, the object is set to undefined. A NULL-pointer as parameter is not permitted!

ATimestamp& operator= (const std::string& time) - Assignment-operator from a const string. The timestamp must be passed as Day.month.year hour:minute:second. If the string does not represent a valid timestamp, the object is set to undefined.

ATimestamp& operator= (const struct tm& time) - Assignment-operator; assigns the values from the passed struct tm.

ATimestamp& operator= (const time_t stamp) - Assignment-operator; sets the time to the value represented by the parameter.

static ATimestamp now () - Returns a timestamp-object representing the current time.

In- & output, conversion

Calculation

ATimestamp& operator+= (const ATimestamp& rhs) - Adds another timestamp-value to the object. An undefined timestamp is treated as "1.1.1900 0:00:00"; if both objects are undefined, the result is undefined. Overflows are corrected.

ATimestamp& operator-= (const ATimestamp& rhs) - Substracts another timestamp-value to the object. An undefined timestamp is treated as "1.1.1900 0:00:00"; if both objects are undefined, the result is undefined. Underflows are corrected.

ATimestamp& operator+ (const ATimestamp& lhs, const ATimestamp& rhs) - Returns the addition of two timestamp-values. An undefined time is treated as "1.1.1900 0:00:00"; if both objects are undefined, the result is undefined. Overflows are corrected.

ATimestamp& operator- (const ATimestamp& lhs, const ATimestamp& rhs) - Returns the substraction of two timestamp-values. An undefined time is treated as "1.1.1900 0:00:00"; if both objects are undefined, the result is undefined. Underflows are corrected.

ATimestamp& add (char Day, char Month = 0, int Year = 0, char Hour = 0, char minute = 0,char second = 0) - If this is not undefined, the passed values are added. Overflows are corrected. The result is returned.

ATimestamp& sub (char Day, char Month = 0, int Year = 0, char Hour = 0, char minute = 0,char second = 0) - If this is not undefined, the passed values are substracted. Underflows are corrected. The result is returned.

Comparison

bool operator== (const ATimestamp& other) - Compares two timestamp-objects and returns true, if they are equal (either the same date/time or both undefined).

bool operator!= (const ATimestamp& other) - Compares two timestamp-objects and returns true, if they are not equal (different date/times and not both undefined).

bool operator< (const ATimestamp& other) - Compares two timestamp-objects and returns true, if this is "older" (further in the past) than other (e.g. "1.1.1900 01:01:01" < "1.1.1900 02:02:02" == true). Undefined values are considered as older than any other time.

bool operator> (const ATimestamp& other) - Compares two timestamp-objects and returns true, if this is "younger" (later on a time-line) than other (e.g. "1.1.1900 01:01:01" < "1.1.1900 02:02:02" == true). Undefined values are considered as older than any other time.

bool operator<= (const ATimestamp& other) - Compares two timestamp-objects and returns true, if this is "older" (further in the past) than other or equal to it (e.g. "1.1.1900 01:01:01" < "1.1.1900 02:02:02" == true). Undefined values are considered as older than any other time.

bool operator>=(const ATimestamp& other) - Compares two timestamp-objects and returns true, if this is "younger" (later on a time-line) than other or equal to it (e.g. "1.1.1900 01:01:01" < "1.1.1900 02:02:02" == true). Undefined values are considered as older than any other time.

long compare (const ATimestamp& other) - Returns the (approximated) difference in days, or - if the day is equal - in seconds between two timestamps. If both objects are undefined, those difference is "0", if only this is undefined the result is MINLONG, if only other is undefined MAXLONG is returned (-> undefined times are considered as (incredible) old).

Utility-functions

virtual bool minAdapt () - Corrects the object after underflows. If the the object is integer after the operation, true is returned (else false).

virtual bool maxAdapt () - Corrects the object after overflows. If the the object is integer after the operation, true is returned (else false).

virtual int checkIntegrity () const - Checks if this object represents a valid timestamp. Note: Even undefined times have valid values!
