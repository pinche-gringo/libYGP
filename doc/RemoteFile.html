Struct RemoteFile

Derived from: File.html
Derived by: none

#include <RemoteFile.h>

Description

Class representing a file on a remote server with simple management-functions.

This class offers the same functionality like its parent although the file is actually stored on a remote computer.

This class is not really (very) useful by itself but is designed to be used by a RemoteDirSearch-object.
