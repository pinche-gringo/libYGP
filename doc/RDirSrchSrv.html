Class RemoteDirSearchSrv

#include <RDirSrchSrv.h>

Description

Class to search for files in a certain directory. This is the server for the RemoteDirSearch-class.

Supported commands are:

  • Check="<file>"
  • Find="<file>";Attr=<attr>
  • Next
  • End

Possible answers to those commands are:

  • RC=<status>[;E=<errortext>]
  • RC=0;File="<file>";Size=<size>;Time=<timestamp>;Attr=<attributes>

RC=0 always means, that the commands completed successfully. This is also true for the "Check"-command; RC=0 means in that case, that the file has been found!
